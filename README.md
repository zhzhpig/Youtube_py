# Youtube_py
<h3>利用谷歌API爬取youtube视频评论，生成表格；</h3>
<p>对表格提供汇总和分析功能;</p>
<p>可进行主题模型分析、语义网络分析:</p>
<ul>
  <li>LDA主题模型</li>
  <li>NMF主题模型</li>
  <li>networkx语义网络</li>
</ul>
<p>可生成可视化图表:</p>
<ul>
  <li>主题分析的词语权重图或html</li>
  <li>语义网络图</li>
  <li>情感类别饼图</li>
  <li>情感极性饼图</li>
  <li>词云图</li>
</ul>
